Angular 1.5 application with ES6 and Components Tutorial

Article: https://thinkster.io/angularjs-es6-tutorial

Install:
1. Clone repo
2. `npm install`
3. `gulp`

Make sure you have gulp installed globally (`npm install -g gulp`)
